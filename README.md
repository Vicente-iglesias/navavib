# Navavib — Landing

Landing page para [navavib.com](https://navavib.com): diseñador y desarrollador potenciado por IA.

## Stack

- Vite + React + TypeScript
- Tailwind CSS v4
- GSAP + ScrollTrigger + Lenis
- Three.js + React Three Fiber

## Desarrollo

```bash
npm install
npm run dev
```

## Producción

```bash
npm run build
npm run preview
```

## Despliegue

Ver [DEPLOY-COOLIFY.md](./DEPLOY-COOLIFY.md) para publicar en Coolify con Docker.

## Contenido

Edita textos e imágenes en `src/data/content.ts` y sustituye assets en `public/`.
